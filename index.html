Free and open source!

Choban is completely free and open source. Github repo can be found on top panel.

Easy to use

Choban is easy to use. No more complicated commands or gui's for installation of softwares.

Fast

Choban is fast because it reads plain JSON manifests that describe how to install a program and its dependencies.

Easy to create and publish packages.

Choban has a package creator to help you with package creation scenario.

Openmind

Choban listens what their users want. This makes Choban more up-to-date and user friendly application.

Secure!

Choban does not publish packages from users directly instead we are inspecting every application. We also have checksum for each application.
