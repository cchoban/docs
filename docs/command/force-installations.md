#### Force installations even if software is already installed.

## Usage
```
chob install <pkg> <pkg2> <pkg3> --force
```
you may use like this if you want
```
chob -S <pkg> <pkg2> <pkg3> ---force
```
## Examples
```
chob -S yarn git firefox ---force
```
